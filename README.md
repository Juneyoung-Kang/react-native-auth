# ReactNativeAuth
React Native with Server Side Authentication Open Source. React Native, Redux, Sails, etc.

## Installation
```
$ git clone https://github.com/Juneyoung-Kang/react-native-auth.git
$ cd react-native-auth
$ yarn
$ react-native run-ios
```
